# Number Guessing Game

A simple **number guessing game** built in Python. The player tries to guess a randomly generated number within a certain range, receiving hints if their guess is too high or too low.

---

## Features

- Random number generation
- User input validation
- Feedback on guesses (too high, too low, or correct)
- Tracks number of attempts

---

## How to Play

1. Run the `number_game.py` script in Python.
2. The program will generate a random number.
3. Enter your guesses until you find the correct number.
4. The program will tell you if your guess is **too high** or **too low**.
5. Once you guess correctly, the program displays your total attempts.
